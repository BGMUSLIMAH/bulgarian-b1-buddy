# Български Trainer B1

A progressive web app (PWA) for learning Bulgarian — built for foreigners living or working in Bulgaria.

## Features

- 600+ vocabulary words across real-life topics (housing, health, university, immigration, market)
- 250+ verbs with present, past and future tense conjugations
- Multiple choice quiz with instant feedback
- Daily practice sessions
- Listening exercises with Bulgarian TTS
- Speaking practice
- Reading comprehension with real-life scenarios
- Full 60-question CEFR placement test (A1–B2)
- Progress tracking with mastery system
- Offline-ready PWA — installable on Android and iOS
- Aviation/mechanical technical vocabulary

## Built With

- React + Vite
- TanStack Router
- Tailwind CSS
- Cloudflare Pages
- Supabase (auth + progress sync)

## Target Users

Foreigners living in Bulgaria — expats, students, workers, and aviation mechanics who need practical Bulgarian fast.
